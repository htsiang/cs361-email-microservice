# cs361-email-microservice
A FastAPI based microservice for CS361 for email reminders with two set API routes - GET and POST, respectively.
## Install dependencies
pip install -r requirements.txt

## Start service
fastapi dev email_microservice.py<br>
The service will run on http://127.0.0.1:8000/

# Communication Contract

## Example call to REQUEST Data
### Endpoint /GET

## Example call to RECIEVE Data
### Endpoint POST/email/task_reminder
